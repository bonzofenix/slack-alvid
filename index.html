Slack Bot for Alvid

Installation

Create a new Bot Integration under services/new/bot.

On the next screen, note the API token.

Deploying to CF

bundle package --all
cf push slack-alvid --no-start
# Slack's Alvid token
cf set-env slack-alvid SLACK_API_TOKEN asas-12345678910-asdasdasdasdasdasdasdasd
cf push slack-alvid

Slack-alvid supports MySql on CF.
